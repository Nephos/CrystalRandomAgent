# CrystalRandomAgent

Random user agents

## Installation


Add this to your application's `shard.yml`:

```yaml
dependencies:
  CrystalRandomAgent:
    github: Nephos/CrystalRandomAgent
```


## Usage


```crystal
require "crystal_random_agent"
CrystalRandomAgent.get_random_user_agent
```
